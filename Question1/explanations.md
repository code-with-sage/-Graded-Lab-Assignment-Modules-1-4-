1.  whoami <br>
    this command display curently login user information.

2. pwd && ls -l<br>
  this command will print working directory and also list all directries in long form where we can 
  see additional information.

3. echo "Linux user environment verified" > user_info.txt<br>
  this command will create a file user_info.txt and add given text in the newly created file. 
   
   cat user_info.txt<br>
   using this command we can verify that the file is created and text is added. 

4. wc user_info.txt<br>
    this command will show  line count, word count, and byte count, of user_info.txt file. 

5. man mkdir<br>
    this comnmand will show user manual with all the options for mkdir command 

6. ls -l ~<br>
    this command will list all files and directories in home directory in long format 

7. grep "admin" log.txt<br>
    this command will search for admin keyword in log.txt file 

8. uname -r<br>
    this command will display the current linux kernal verion 

9. ping www.google.com<br>
   This command checks network connectivity to the specified server www.google.com. 

10. uptime<br>
   This command displays the system uptime, number of logged-in users, and the system load average.
